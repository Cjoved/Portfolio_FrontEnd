PROFILE PHOTO – SAAN ILALAGAY
=============================

1. HERO (avatar, kanan):
   - I-save/copy ang profile image dito:  avatar.png
   - (Same folder as this file – public/)

2. ABOUT (profile, kaliwa ng text):
   - I-save/copy ang profile image dito:  profile.png
   - (Same folder – public/)

Pwede i-copy lang ang iisang image sa dalawang filename (avatar.png at profile.png) kung pareho lang ang gagamitin.

ALISIN ANG BACKGROUND (WHITE)
=============================

Hindi kayang gawin ng code ang actual background removal. Gamitin ang:

• remove.bg  (https://www.remove.bg) – upload photo, download PNG na walang background
• Photopea   (https://www.photopea.com) – free online Photoshop, use Magic Wand o Eraser
• Canva      – upload, “Background remover” tool

Pag may nang transparent background na PNG na, i-save bilang avatar.png at profile.png sa public/ folder.
